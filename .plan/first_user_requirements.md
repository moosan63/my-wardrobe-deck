# ワードローブ管理アプリ

- 自分の所有してる服を管理するためのアプリ
- カードデッキのように自分の洋服を俯瞰してちょっと面白く管理したい

## 機能

- 所持している洋服のCRUD
- 所持している洋服のカード型UIでの一覧
- アウター・トップス・ボトムスのような区別でグルーピングされて一覧化されている
  - 分類は要確認
- 洋服にタグのようなものをつけて、それで分類・フィルターもできる
  - 使用季節, 色, 形(ロングコート等、ある程度プリセット＋自由入力), ブランド, 用途(おしゃれ着、普段着、ワンマイルウェア、寝間着など)

## 技術スタック

| 項目 | 技術 |
|------|------|
| 言語 | TypeScript |
| フレームワーク | HonoX |
| サーバー | Cloudflare Workers |
| データベース | Cloudflare D1 |
| スケジューラー | Cloudflare Workers Cron Triggers |
| スタイリング | Tailwind CSS |
